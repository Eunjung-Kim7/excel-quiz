#N/A
Not Available — 값 없음
🔍 언제 발생? 찾는 값이 범위에 존재하지 않을 때. VLOOKUP·XLOOKUP·MATCH에서 가장 빈번.
=VLOOKUP("홍길동",A:B,2,FALSE) → A열에 "홍길동" 없음
검색값 앞뒤에 공백 있을 때 (눈에 안 보이지만 다른 값)
숫자처럼 보이지만 텍스트로 저장된 셀
✅ 해결법
IFERROR로 감싸기: =IFERROR(VLOOKUP(...),"없음")
TRIM 함수로 공백 제거: =TRIM(A1)
VALUE 함수로 숫자 변환: =VALUE(A1)
XLOOKUP 사용 시 4번째 인수에 대체값 지정
#VALUE!
잘못된 값 형식
🔍 언제 발생? 수식에 숫자가 와야 할 곳에 텍스트가 들어갔을 때.
=A1+B1 → A1이 "100원" 텍스트일 때
날짜처럼 보이지만 텍스트로 저장된 셀 계산
배열 수식에서 크기 불일치
✅ 해결법
셀 서식 확인 → 숫자 형식으로 변경
VALUE 함수: =VALUE(SUBSTITUTE(A1,"원",""))
텍스트 나누기로 데이터 형식 재정의
IFERROR로 감싸서 오류 숨기기
#REF!
Reference — 참조 오류
🔍 언제 발생? 수식이 참조하던 셀·행·열이 삭제됐을 때.
VLOOKUP 범위에서 열 삭제 → 열 번호 인수가 범위 초과
수식이 있는 셀 자신을 직접 참조 (순환 참조)
행/열 삭제 후 수식이 존재하지 않는 셀 참조
✅ 해결법
Ctrl+Z로 즉시 되돌리기
XLOOKUP으로 교체 → 열 삭제에 강함
수식 감사 → 참조 추적으로 원인 셀 파악
#DIV/0!
Division by Zero — 0으로 나누기
🔍 언제 발생? 분모가 0이거나 빈 셀일 때.
=A1/B1 → B1이 0 또는 빈 셀
평균 계산 시 데이터 없는 범위 지정
비율 계산 시 전체 합이 0
✅ 해결법
=IF(B1=0,"―",A1/B1)
=IFERROR(A1/B1,0)
AVERAGEIF로 0 제외 평균 계산
분모 셀에 데이터 입력 여부 사전 확인
#NAME?
함수명·이름 인식 불가
🔍 언제 발생? 함수 이름이 잘못 입력됐거나, 텍스트에 따옴표가 없을 때.
=SSUM(A1:A10) — 오타 (SSUM → SUM)
=VLOOKUP(홍길동,A:B,2,0)
— 텍스트에 따옴표 누락

🔗 셀 참조 방식 비교
방식형식복사 시예시
상대 참조A1행·열 자동 이동=B2-C2
절대 참조$A$1행·열 모두 고정=B2*$D$1
혼합 참조A$1 / $A1한 쪽만 고정=B$2*$A3
💡 F4 전환 순서: A1 → $A$1 → A$1 → $A1 → A1

🏷️
이름 정의
범위에 이름을 붙여 수식을 직관적으로 작성
=단가*(1-할인율)*수량

➕ 합계·통계 함수
➕
SUM / AVERAGE
합계·평균. Alt+= 로 SUM 자동 입력
=SUM(B4:G4) / =AVERAGE(B4:G4)
📊
MAX / MIN / LARGE / SMALL
최대·최솟값, N번째 크고 작은 값
=LARGE(범위, 2) → 2번째로 큰 값
🔢 개수 함수 비교
함수대상활용 예
COUNT숫자 셀만점수 셀 수
COUNTA비어있지 않은 셀출석일 세기
COUNTBLANK빈 셀결석일 세기
COUNTIF단일 조건=COUNTIF(B:B,"서울")
🔣 수학 함수
🔽
INT / ROUND 계열
INT: 무조건 내림 / ROUND(수,n): 반올림
ROUNDUP: 올림 / ROUNDDOWN: 내림
=ROUND(포상금, 0) → 정수 반올림
➗
QUOTIENT / MOD
QUOTIENT: 몫 / MOD: 나머지
=QUOTIENT(2510,50)→50 / =MOD(2510,50)→10
🔢
ROW / SUMPRODUCT
ROW(): 현재 행 번호 / SUMPRODUCT: 배열 곱 후 합산
=SUMPRODUCT(수량범위, 단가범위)
🏆
RANK.EQ / RANK.AVG
RANK.EQ: 동점 동일 순위 / RANK.AVG: 평균 순위
범위는 절대참조($)로!
=RANK.EQ(G3,$G$3:$G$19,0)
🔀 IF 계열 함수
❓
IF 기본 / 중첩 IF
큰 조건부터 체크!
=IF(A1>=90,"A",IF(A1>=80,"B","C"))
📋
IFS
다중 조건. TRUE로 기본값 설정
=IFS(I4>=7,"A",I4>=5,"B",TRUE,"C")
🤝
IF + AND / OR
AND: 모두 만족 / OR: 하나라도 만족
=IF(AND(D4="CR",OR(C4="A",C4="B")),"정상","워크아웃")

표 구조적 참조
